# TM3_Tools 🔧

An all-in-one toolkit for TM3Admin focused on improving productivity and simplifying daily workflows. This project is built to assist with music score (partitur) development, choir administration, content preparation, and various choir-related tasks through practical tools and automation.

This project is also created with the hope that it can support and help fellow team members in their daily work, reduce repetitive tasks, and make processes more efficient and enjoyable. Beyond being just a collection of tools, it is hoped that this project can encourage collaboration, make work feel lighter, and help everyone stay motivated, inspired, and enthusiastic in serving and growing together through choir-related activities.

Currently available:

- Sheet music (partitur) watermark tool

Planned future features:

- Automatic partitur creation/generation
- Partitur creation from song lists
- PDF merging tools
- PDF + PNG/JPG combining tools
- Additional choir-related utilities and workflow tools

This project will continue to grow over time with new ideas and features that can provide practical benefits and make TM3Admin activities more efficient and enjoyable.
